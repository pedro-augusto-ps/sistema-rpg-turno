# Sistema RPG por Turnos

Este projeto permite ao usuário realizar uma breve jogatina contra a máquina, pelo terminal o jogador escolhe entre atacar ou usar uma poção de cura,  com o objetivo de derrotar o monstro

Atráves deste repositório quero registrar meu aprendizado e evolução.

## ESTRUTURA: Sistema separado em dois arquivos
- "funcoes.py" 
- "main.py"
